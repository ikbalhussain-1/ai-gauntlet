# Submitting Your Score

## How to score yourself

Run this from your project root directory:

**macOS / Linux:**
```bash
bash scripts/check.sh
```

**Windows:**
```powershell
.\scripts\check.ps1
```

The script will:
- Check every part of your implementation
- Run your tests
- Test your AI integration against 5 labelled reviews
- Print your score out of 100
- Save a report to `SCORE_REPORT.md`

---

## How to submit to the leaderboard

1. Take a **screenshot** of your terminal showing the final score
2. Post it in the group chat with your name and score
3. First to **80+ points** wins the day

---

## Score Guide

| Score | Badge | What it means |
|-------|-------|---------------|
| 90–100 | 🏆 Elite | Stage 7 — you're running a fleet |
| 75–89  | 🥇 Stage 6 | Multi-agent, minimal oversight |
| 55–74  | 🥈 Stage 5 | CLI, single agent, strong output |
| 35–54  | 🥉 Stage 4 | Good progress, tests need work |
| 0–34   | 🔧 Stage 3 | Where we started — keep going |

---

## The score doesn't capture everything

Some things the script can't measure — but you should reflect on:

- Did you read any implementation code today, or only diffs?
- Did Claude catch any bugs you would have missed?
- What would this have taken you without AI?

The real leaderboard is internal. This is just the warmup.
